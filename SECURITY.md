# Security and Confidentiality

Репозиторий содержит финансовые допущения и сведения об операционной деятельности.

## Не размещать в репозитории

- персональные данные работников;
- банковские реквизиты;
- пароли, токены и ключи API;
- полные договоры с конфиденциальными условиями;
- медицинские документы;
- исходные документы с подписями;
- данные, не предназначенные для участников проекта.

## Разрешённый формат источников

В документации фиксируются название источника, дата, владелец, статус проверки и ссылка на закрытое хранилище без копирования конфиденциального содержимого.
