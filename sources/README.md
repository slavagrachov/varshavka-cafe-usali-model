# Реестр первичных источников

Первичные документы хранятся вне GitHub в закрытом Google Drive:

- схема помещений;
- проект договора аренды;
- проект договора питания;
- связанные документы объекта.

Файлы не копируются в репозиторий из-за конфиденциальности. В репозитории фиксируются только извлечённые параметры, источник и статус проверки.
