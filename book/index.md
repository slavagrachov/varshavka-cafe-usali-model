# Финансовая модель кафе VARSHAVKA

Эта книга объединяет описание проекта, устройство Excel-модели, методики расчёта, сценарии, финансовые результаты и правила управления изменениями.

## Назначение

Jupyter Book служит публичной и версионируемой документацией финансовой модели. Расчётным источником остаются утверждённые Excel-файлы в репозитории и GitHub Releases.
