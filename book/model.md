# Финансовая модель

## Архитектура Excel-книги

| Лист | Назначение |
|---|---|
| `00_РЕЗЮМЕ` | ключевые показатели и итоги |
| `01_ВВОД` | единый реестр входных данных |
| `02_КАЛЕНДАРЬ` | календарь, праздники и банкеты |
| `03_ДОХОДЫ` | расчёт выручки по каналам |
| `04_СЕБЕСТОИМОСТЬ` | продуктовая себестоимость и прямые расходы |
| `05_ФОТ` | штат, зарплата и начисления |
| `06_OPEX_USALI` | операционные расходы |
| `07_PNL_НАЛОГИ` | P&L, налоги и чистый результат |
| `08_ПРОВЕРКИ` | контроль качества модели |
| `09_СРАВНЕНИЕ_S03` | сравнение S03 с базовой моделью |
| `10_СРАВНЕНИЕ_S02_S03` | сравнение S02 и S03 |

## Доходы

Выручка рассчитывается по направлениям: a-la carte, бизнес-ланч, доставка, самовывоз, гостиничный канал, ужины и банкеты.

## Себестоимость

Food cost a-la carte равен 30%. Food cost ужина гостей гостиницы связан с этим же параметром. Для бизнес-ланча применяется эффективный food cost с учётом скидки.

## ФОТ и OPEX

ФОТ выделен отдельно от OPEX. Операционные расходы сгруппированы по управленческой логике, адаптированной на основе USALI.

## Налоги и результат

Модель рассчитывает результат кафе, результат банкетов, совокупную прибыль до налога, минимальный налог и чистый результат проекта.
