# Методика себестоимости

| Направление | Базовый COGS |
|---|---:|
| А-ля карт | 30% |
| Бизнес-ланч | 40% от фактической выручки |
| Доставка | 30% |
| Самовывоз | 42,857% |
| Гостиница | 50% |
| Банкеты | 40% |

Дополнительно для кафе:

- списания и порча: 1,5% продуктовой себестоимости;
- инвентаризационные расхождения: 0,5%;
- доставщик: 200 руб. на заказ;
- упаковка доставки: 65 руб. на заказ;
- упаковка самовывоза: 40 руб. на заказ;
- эквайринг: 1,5% безналичной выручки.

Оценочные ставки должны быть заменены фактическими данными после начала работы.
