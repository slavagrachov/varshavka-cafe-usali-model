# Методика налогов

## УСН

- объект: «Доходы минус расходы»;
- ставка: 15%;
- минимальный налог: 1% доходов.

```text
Расчётный УСН = MAX(0; налоговая база × 15%)
Минимальный налог = доходы × 1%
Налог к уплате = MAX(расчётный УСН; минимальный налог)
```

Налог рассчитывается консолидированно по проекту. Для управленческого результата он распределяется между кафе и банкетами пропорционально выручке.

## Открытый вопрос

НДС установлен в 0% и должен быть подтверждён отдельно с учётом режима УСН, оборота и даты возникновения обязанности.
