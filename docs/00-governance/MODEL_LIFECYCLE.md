# Жизненный цикл модели

```text
Замысел
→ сбор данных
→ аудит данных
→ методика
→ архитектура
→ Excel-реализация
→ проверка
→ утверждение
→ эксплуатация
→ plan/fact
→ журнал изменений
→ рефакторинг
→ новый релиз
```

## Статусы

| Статус | Значение |
|---|---|
| Draft | формируется |
| Validated | прошёл технические проверки |
| Approved | утверждён владельцем |
| Operational | используется для управления |
| Superseded | заменён новой версией |
| Archived | сохранён для истории |

Текущая версия `v0.1.0` — рабочая операционная база с подтверждёнными и оценочными inputs.
