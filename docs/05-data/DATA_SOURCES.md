# Источники данных

| Источник | Использование | Статус |
|---|---|---|
| Экспертные ответы владельца | чеки, гости, COGS, штат, OPEX | основной |
| Схема помещений | площадь №19, №20, №39 = 80,9 м² | подтверждено |
| Проект договора аренды | страхование и эксплуатационные обязанности | использовано методически |

| Налоговые источники ФНС | УСН и НДС | требуют проверки на дату релиза |
