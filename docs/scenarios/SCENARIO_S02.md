## Цель

Создать альтернативный операционный сценарий, в котором проведение банкета не приводит к полной остановке дневной деятельности кафе.

## Правила банкетного дня

| Канал | Правило S02 |
|---|---:|
| Завтраки гостиницы | 100% обычного объёма |
| Бизнес-ланч | 100% обычного объёма в рабочие дни |
| А-ля карт | 20% нормы соответствующего дня; работа до 17:00 |
| Доставка | 100% обычного объёма |
| Самовывоз | 100% обычного объёма |
| Ужины гостиницы | 50% от количества завтраков |
| Банкет | согласно календарю базовой модели |
