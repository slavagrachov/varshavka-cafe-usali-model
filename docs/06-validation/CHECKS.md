# Контрольные проверки

В Excel реализованы проверки:

- 365 календарных дней;
- 12 месяцев;
- 60 банкетов;
- 0 банкетов в сентябре–октябре;
- 42 банкета в пятницу–воскресенье;
- 18 банкетов в понедельник–четверг;
- 7 300 гостиничных заказов;
- нулевые обычные продажи кафе в банкетный день;
- связь доходов;
- связь маржинального дохода;
- связь ФОТ;
- связь OPEX;
- распределение налога;
- консолидация чистой прибыли.
