# Руководство пользователя

## Что можно менять

Изменяйте только входные значения на листе `01_ВВОД`. Синие ячейки предназначены для ввода; жёлтые/оранжевые указывают на допущения.

## Последовательность обновления

1. Сохраните копию текущего релиза.
2. Обновите input и источник.
3. Проверьте календарь банкетов.
4. Откройте `08_ПРОВЕРКИ`.
5. Убедитесь, что статусы `OK`.
6. Проверьте `00_РЕЗЮМЕ`.
7. Обновите changelog и создайте новую версию.

## Запрещено

- вручную менять итоговые строки P&L;
- вставлять значения поверх формул;
- менять структуру листов без ADR;
- удалять контрольные проверки;
- заменять релизный файл без новой версии.
