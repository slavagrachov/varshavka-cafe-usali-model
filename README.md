# VARSHAVKA Cafe USALI Model

Операционная финансовая модель кафе **VARSHAVKA** с отдельным финансовым контуром банкетов, классификацией операционных расходов по логике USALI и помесячным горизонтом **сентябрь 2026 — август 2027**.

> Репозиторий является самостоятельным проектом и не связан расчётной логикой с `slavagrachov/varshavka-finmodel`.

### Ключевые правила S02

- завтраки гостиницы в банкетный день — 100%;
- бизнес-ланч — 100%;
- а-ля карт — 20% нормы соответствующего дня, работа до 17:00;
- доставка — 100%;
- самовывоз — 100%;
- ужины гостиницы — 50% от количества завтраков;
- банкет — согласно календарю базовой модели;
- постоянные расходы и штат не изменяются.

## Быстрый старт

1. Для базового варианта откройте файл из `models/`.
2. Для S02 откройте файл из `models/scenarios/S02/`.
3. Изменяйте только синие input-ячейки на листе `01_ВВОД`.
4. Контролируйте лист `08_ПРОВЕРКИ`: все статусы должны быть `OK`.
5. Для сравнения используйте лист `09_СРАВНЕНИЕ_S02`.
6. Любое изменение выполняйте через Issue, отдельную ветку и Pull Request.

## Ограничения

- НДС требует отдельного окончательного решения.
- Производственный календарь и переносы праздников 2027 года требуют актуализации.
- Ряд OPEX и параметры ужина являются рабочими допущениями.
- Инвестиции, оборотный капитал, cash flow запуска и окупаемость не рассчитаны.
- Штат и графики требуют операционной проверки на сменность и замещение.
