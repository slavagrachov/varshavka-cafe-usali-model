# VARSHAVKA Cafe USALI Model

Операционная финансовая модель кафе **VARSHAVKA** с горизонтом сентябрь 2026 — август 2027.

## Jupyter Book

Документация модели подготовлена в формате Jupyter Book и публикуется через GitHub Pages:

[Открыть Jupyter Book](https://slavagrachov.github.io/varshavka-cafe-usali-model/)

Исходные страницы книги находятся в папке `book/`. После изменения книги и объединения Pull Request с `main` GitHub Actions автоматически пересобирает сайт.

## Работа с моделью

1. Изменяйте только input-ячейки листа `01_ВВОД`.
2. После изменений контролируйте лист `08_ПРОВЕРКИ`.
3. Любое изменение оформляйте через Issue, отдельную ветку и Pull Request.
4. Скрипт `scripts/s03_model_builder.py` воспроизводимо строит S03 из утверждённого S02.

## Ограничения

- НДС требует окончательного решения;
- часть OPEX требует документального подтверждения;
- средний чек ужина требует бизнес-подтверждения;
- инвестиции, Cash Flow и окупаемость не рассчитаны.
